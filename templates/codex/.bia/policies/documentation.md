# Política de documentación viva

- OpenSpec describe el cambio; `docs/project/` describe el sistema vigente; Engram conserva contexto.
- Una decisión significativa de arquitectura, tecnología, seguridad, persistencia, compatibilidad o reversión costosa exige ADR enlazado.
- `tasks.md` incluye actualizaciones documentales con destino observable.
- `verify` valida enlaces, ADR, documentación y equivalencia híbrida; `archive` se bloquea ante pendientes.
