# Registro de decisiones

Cree un ADR para cambios de arquitectura, tecnología, seguridad, persistencia, compatibilidad o reversión costosa. Numere con `max(ADR-NNNN)+1`, nunca reutilice números y enlace sustituciones en ambos documentos.

Estados permitidos: `Propuesto`, `Aceptado`, `Rechazado` y `Sustituido`.
