# Continuar un cambio SDD

Lea la gobernanza y ejecute el preflight B-IA con el estado estructurado del dispatcher de Gentle AI. Delegue exclusivamente la fase indicada por `nextRecommended`; nunca infiera routing desde texto conversacional.

Tras la fase, valide artefacto, documentación y persistencia híbrida. Un bloqueo o código distinto de cero detiene las fases dependientes. Esta skill no mantiene estado paralelo, no escribe `state.yaml` y no aprueba su propia salida.
