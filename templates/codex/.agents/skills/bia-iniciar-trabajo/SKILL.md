# Iniciar trabajo

Esta skill se activa con `iniciar trabajo` o `$bia-iniciar-trabajo`; ambas expresiones delegan este mismo workflow.

Resuelva el repositorio y reúna Git, OpenSpec, Engram y el dispatcher de Gentle AI. Declare cada fuente inaccesible como `unknown`. Presente trabajo vigente, bloqueos, pendientes y prioridades según `.bia/policies/daily-work.md`. No ejecute tareas, no cambie fases y no cree commits.
